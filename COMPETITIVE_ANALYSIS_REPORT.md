## 5. World-Class Enhancement Plan (Prioritized by Impact × Effort)

### 🔴 CRITICAL — Do These First (High Impact, Low-Medium Effort)

#### 5.1 Add Live Audio Demo Player (Impact: 9/10, Effort: Medium)
**Problem:** 3.0/10 on feature demonstration — visitors can't hear Katie before signing up.  
**Solution:** Add an audio player section showing a 60-90 second sample call.
- Record a demo: "Katie answering a boiler repair enquiry"
- Use Web Audio API or HTML5 audio with custom styled player
- Place below hero or in "Hear It in Action" section
- Add transcript below player for accessibility
- **Expected conversion lift:** +15-25% (based on TR's strong demo placement)

#### 5.2 Add Homepage Comparison Table (Impact: 8/10, Effort: Low)
**Problem:** No answer to "why not just use voicemail or a VA?"  
**Solution:** Add a swipeable/scrollable comparison table.
| Feature | whoza.ai | Voicemail | Human VA | Traditional Answering |
|---------|----------|-----------|----------|----------------------|
| 24/7 answering | ✅ | ❌ | ❌ | Partial |
| Cost/month | £59-249 | £0* | £800-1200 | £100-300 |
| AI visibility scores | ✅ | ❌ | ❌ | ❌ |
| Review collection | ✅ | ❌ | ❌ | ❌ |
| WhatsApp summaries | ✅ | ❌ | ❌ | ❌ |

*"Voicemail appears free but costs £4,200/year in missed jobs" — borrow TR's framing

#### 5.3 Add WhatsApp Chat Widget (Impact: 7/10, Effort: Low)
**Problem:** No live support channel. Visitors with questions bounce.  
**Solution:** Floating WhatsApp button (like TR's).
- Link to `wa.me` with pre-filled message
- Style as green bubble with WhatsApp icon
- Show on mobile (bottom right, above sticky CTA)
- Optional: show " typically replies in 5 minutes" tooltip

#### 5.4 Real Testimonial Photos + Outcome Tags (Impact: 7/10, Effort: Medium)
**Problem:** Colored circles feel generic. TR's named quotes with locations feel authentic.  
**Solution:**
- Replace colored circles with real trade person photos (stock or request from actual users)
- Add outcome tags: "+3 jobs/month", "Diary fully booked", "Family time back"
- Keep the metric badges we already have (+3 jobs/month, etc.)
- Add location + trade to each testimonial (we have this partially)

#### 5.5 Fix Cookie Consent Styling (Impact: 5/10, Effort: Low)
**Problem:** Generic unstyled dialog breaks premium immersion immediately on first visit.  
**Solution:** Style to match design system — dark navy background, Inter font, rounded corners, subtle shadow.

- "Get your full score — takes 2 minutes" → CTA to signup
- This is a lead magnet that proves the value before commitment
