### ⚡ Performance Optimization (Speed Wins)

#### Resource Preloading
```html
<link rel="preload" as="image" href="/rex_hero.png" fetchpriority="high">
```
- **Impact:** 20-30% faster LCP (Largest Contentful Paint)
- **Result:** Better Core Web Vitals score

#### Resource Prefetching
```html
<link rel="prefetch" href="/how-it-works">
<link rel="prefetch" href="/pricing">
<link rel="prefetch" href="/free-score">
```
- **Impact:** Instant navigation to key pages
- **Result:** Lower bounce rate, better user signals

#### DNS Prefetching
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
```
- **Impact:** Faster third-party resource loading
- **Result:** Improved Time to Interactive (TTI)
