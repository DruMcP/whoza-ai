🚀 GitHub Actions Deployment Assistant

Deploy your automated daily KPI reports in 5 minutes

1 Open Your GitHub Repository

Click the button below to open your repository in a new tab:

📂 Open Repository
💡 Tip: Keep this page open in one tab and work in the GitHub tab. Come back here for each step.

2 Create New Workflow File

In your GitHub repository:

  • Click the "Add file" dropdown button (top right)
  • Select "Create new file"
📍 Location: You should see a text input for the filename at the top of the page.

3 Enter Workflow File Path

In the filename field, copy and paste this exact path:

.github/workflows/daily-kpi-reports.yml
⚠️ Important: The path must be exact, including the dots and slashes. GitHub will automatically create the directories.

4 Copy Workflow Content

Click the button below to view and copy the workflow file content:

Loading workflow content...
📋 Instructions:
  1. Click "Show Workflow Content" above
  2. Click the "Copy All" button
  3. Go back to GitHub tab
  4. Paste into the file editor (Ctrl+V or Cmd+V)

5 Commit the File

Scroll to the bottom of the GitHub page and:

  1. In the "Commit message" field, paste this:
feat: Add GitHub Actions workflow for automated daily KPI reports
  1. Leave "Commit directly to the main branch" selected
  2. Click the green "Commit new file" button

6 Verify Workflow Deployment

Let's confirm the workflow was deployed successfully:

⚙️ Open Actions Tab

✅ Success Indicators:

  • You should see "Daily KPI Reports" in the workflows list (left sidebar)
  • Status should show "Waiting for first run" or similar
  • No error messages displayed

7 Enable Workflow Permissions

The workflow needs permission to commit reports to your repository:

🔒 Open Actions Settings
📝 Instructions:
  1. Scroll down to "Workflow permissions"
  2. Select "Read and write permissions"
  3. Check "Allow GitHub Actions to create and approve pull requests"
  4. Click "Save"

8 Test the Workflow

Let's run the workflow manually to verify everything works:

▶️ Open Workflow
🎯 Test Instructions:
  1. Click the "Run workflow" button (right side)
  2. Leave the date field empty (uses today)
  3. Leave "Send email notification" unchecked (for now)
  4. Click the green "Run workflow" button
  5. Wait 2-3 minutes for completion
  6. Refresh the page to see results

✅ Expected Results:

  • Workflow run appears with green checkmark ✓
  • Job summary shows KPI metrics
  • New report file in reports/ directory
  • Artifacts available for download

9 🎉 Deployment Complete!

✅ Your automated daily KPI reports are now live!

What happens next:

  • 📊 Reports generate automatically every day at 9:00 AM GMT
  • 📈 Tracks Engagement (+55% target), Trust (+40% target), Conversion (+45% target)
  • 💾 Reports auto-commit to your repository
  • 📦 90-day artifact retention
  • 🔄 You can trigger manual reports anytime
