# Process — New Project Kickoff

## Trigger
New initiative approved by Dru or agent team.

## Steps
1. **Define objective**
   - One-line goal
   - 3–5 key results (specific, measurable)

2. **Create project note**
   - Use [[Templates/project]] in `01-Projects/`
   - Fill frontmatter: status, owner, priority
   - Link to triggering decisions or research

3. **Assign owner**
   - Primary agent responsible
   - Secondary agents supporting
   - Human (Dru) oversight points

4. **Set timeline**
   - Kickoff date
   - Review milestones
   - Target completion

5. **Define resources**
   - Budget
   - Tools needed
   - Dependencies on other projects

6. **Update indexes**
   - Add to [[Index — Projects]]
   - Link from relevant decision/research notes

7. **Announce**
   - Notify relevant agents
   - Update any project status dashboards

## Escalation
- If project conflicts with existing work → Agent-1 prioritization
- If project needs resources not available → Dru decision
- If project scope unclear → Refine in inbox before kickoff
